Anotações

    Principais

  • Para iniciar o React digitace no terminal npx create-react-app meu-projeto

    • JSX/HTML

  • JSX é o HTML do React

    • CSS

  • Lembrar de quando for colocar class em algum elemento, utilizar className

  • O index.css vai ser sempre o CSS Global do projeto

  • Para fazer o CSS de um componente expecifico deve criar um arquivo .css com o nome de NomeDoComponente.module.css

    • Fragmentos

  • É usado no caso de ser apenas um item separado, exemplo, uma lista, quero criar apenas o item da lista separado, não preciso criar a div como Pai, apenas uso <>

    • Tipos de props

  • Para importar proto-type coloca import ProtoType from "proto-type" no item que vai ter um tipo

    • Eventos

  • Eventos são como no JavaScript puro, mas são adicionados ao elemento diretamente no código

    • Formulario

  • Na função de Submit de um formulario, sempre utilizar e.preventDefault()

    para não atualizar a pagina e ver se a função está funcionando

    • Hooks (pegar e armazenar dados do Form)

    useState:

  • Sempre importar no arquivo que será usado com import { useState } from 'react'

  • Cria-se duas contantes para cada dado que vai ser manipulado como por exemplo const [name. setName] = useState() nesse caso, poderia ser manipulado o name recebido de um imput por exemplo

    • Mapear Arrays

  • Usa-se por exemplo itens.map para mapear os itens de uma lista. Exemplo no arquivo "Outraslistas"

  • O if e else no React é utilizado com, exemplo:
    {itens.legth > 0 ? (
    "O que vai acontecer no if" ) : (
    "O que vai acontecer no else" )
    }

    Exemplo no arquivo "Outraslistas"

    • Router React

  • Serve para navegar entre paginas sem precisar recarregar a pagina como é por padrão

  • Para Instalar o Router se digita no prompt: npm install react-router-dom

  • Para importar no App.js se import {BrowserRouter as Router, Switch, Router, Link} from 'react-router-dom'

  • No App.js se coloca < Router > como Pai
