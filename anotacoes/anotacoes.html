Anotações

    Principais

  • Para iniciar o React digitace no terminal npx create-react-app meu-projeto

    • JSX/HTML

  • JSX é o HTML do React

    • CSS

  • Lembrar de quando for colocar class em algum elemento, utilizar className

  • O index.css vai ser sempre o CSS Global do projeto

  • Para fazer o CSS de um componente expecifico deve criar um arquivo .css com o nome de NomeDoComponente.module.css

    • Fragmentos

  • É usado no caso de ser apenas um item separado, exemplo, uma lista, quero criar apenas o item da lista separado, não preciso criar a div como Pai, apenas uso <>

    • Tipos de props

  • Para importar proto-type coloca import ProtoType from "proto-type" no item que vai ter um tipo
